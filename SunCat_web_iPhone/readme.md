# SUNCAT OS

아이폰(iOS 18) 스타일을 모티브로 제작한 개인 홈페이지 프로젝트입니다.

## Features

- iOS 스타일 잠금화면
- 위로 스와이프하여 잠금 해제
- 실시간 시계 및 날짜 표시
- Dynamic Island UI
- Glassmorphism 디자인
- iOS 스타일 홈화면
- 사진 앱(Gallery)
- 사진 확대 보기
- 좌우 사진 이동
- 모바일 및 데스크탑 지원
- 반응형 UI

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Preview

SUNCAT OS 느낌의 개인 포트폴리오 및 갤러리 홈페이지입니다.
